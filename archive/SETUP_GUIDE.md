# Web Developer & Contributor Setup Guide

This guide provides step-by-step instructions for setting up the WRIT20833 portfolio template as a web developer and inviting contributors. It does NOT cover uploading or editing content.

## 1. Web Developer Setup

### Prerequisites
- Install [Git](https://git-scm.com/) on your computer.
- Install a code editor, such as [VS Code](https://code.visualstudio.com/).


### Clone the Repository Using GitHub Desktop
1. Open [GitHub Desktop](https://desktop.github.com/).
2. Click `File > Clone Repository`.
3. Search for `TCU-DCDA/WRIT20833-portfolio-template` or paste the repository URL: `https://github.com/TCU-DCDA/WRIT20833-portfolio-template`.
4. Choose a local path and click `Clone`.
5. Click `Open in Visual Studio Code` from GitHub Desktop, or open VS Code and select the cloned folder.


### Open in VS Code
1. Launch VS Code.
2. Open the project folder (`WRIT20833-portfolio-template`) if not already opened from GitHub Desktop.

### Install Live Server Extension (Optional)
- For live preview, install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code.

## 2. Inviting Contributors

### Add Contributors on GitHub
1. Go to the repository page on GitHub.
2. Click on `Settings` > `Collaborators`.
3. Add the GitHub usernames of contributors.

### Share Setup Instructions
- Direct contributors to this setup guide for cloning and opening the project.

### Contributor Access
- Contributors will be able to clone, pull, and push changes after accepting the invitation.
